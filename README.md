# dheeraj-reddy.in

Personal blog and portfolio — notes on embedded systems, Linux kernel work, reverse engineering, and things that caught my curiosity.

**Live site:** [dheeraj-reddy.in](https://dheeraj-reddy.in)

## What's here

- **Blog posts** — embedded systems, Linux, reverse engineering
- **Conference Talks** — slides and recordings from talks I've given
- **Photography** — a collection of shots from travels and everyday life
- **Contact** — a contact form (no email address exposed)

## Built with

If you want to build something similar, the easiest way to get started is the [Chirpy Starter template](https://github.com/cotes2020/chirpy-starter) — fork it, configure `_config.yml`, and you're writing in minutes.
